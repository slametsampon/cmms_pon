CMMS - Integrated System

CMMS (Computer Maintenance Management System) adalah sebuah system yang dirancang untuk pengelolaan (manage) seluruh kegiatan maintenance (pemeliharaan) pabrik. System ini dibuat web-base sehingga dilakukan akses via internet. Adapun keunggulan yang diperoleh dengan CMMS ini adalah :

  • Paperless
  • Traceability pada setiap kegiatan
  • OTE : One Time Entry
  • Minimize pekerjaan administrasi dengan system OTE

Work Order - Permintaan Pekerjaan

Module MWO adalah modul yang menangani permintaan pekerjaan (work order), meliputi hal hal berikut:

  • Membuat permintaan pekerjaan
  • Approval herarcy
  • Tracing/Tracking Work Order
  • Action reporting
  • Searching Work Order
  • Summary Work Order

PM PdM - Preventive and Predictive Maintenance

Module PPdM adalah module yang menangani Preventive dan Predictive Maintenance termasuk di dalamnya kalibrasi, meliputi hal-hal berikut ini :

  • Set-up dan konfigurasi
  • Scheduling
  • Automatic Work Order
  • Action reporting
